Smart Parksystem

Hardware-Komponenten
  • 2× Arduino Uno
  • 2× Ultraschall-Sensor (HC-SR04)
  • 1× Servo-Motor (Schranke)
  • 1× LCD-Display (I²C)
  • 1× Taster (Schranken-Öffner)
  • Kabel, Breadboard, USB
Funktionsweise
  • Sensoren überwachen Parkplatzbelegung
  • LCD zeigt freie / belegte Plätze
  • Schranke öffnet nur bei freiem Platz
  • Zwei Arduinos kommunizieren per Analogpin
  • Web-Interface via USB Serial

Web-Interface herunterladen

Lade die HTML-Datei herunter, öffne sie lokal in Chrome oder Edge und verbinde deinen Arduino per USB – dann siehst du die freien Parkplätze live.

⚠ Benötigt Google Chrome oder Microsoft Edge
⚠ Arduino muss per USB angeschlossen sein
⚠ Arduino-Code muss laufen & Serial auf 9600 Baud
